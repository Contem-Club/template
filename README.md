# template app

A template flutter application

## Getting Started

This project is a starting point for a Flutter application.
Using the @override in the build method allows for hot reloads

## Tips
- Use https://appicon.co to generate icon files for android and iOS.
  Search the corresponding folders in the android and ios subolders of the project
  To simply replace the existing icon files

- Right click on the ios, the use flutter -> open in xcode to open the project in xcode
  Click on the project, check the "signing" section and either add your account or (if
  already done) select the proper team. This then allows to also deploy to an physical
  iOS device (see lesson
  https://www.udemy.com/course/flutter-bootcamp-with-dart/learn/lecture/14512084#overview)
